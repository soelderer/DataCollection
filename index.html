The following rules apply to the Cognitive-Affective Map you should draw:
  • Draw a minimum of 24 concepts in total.
  • Use no more than three words per concept.
  • Connect all the drawn concepts with each other.
If you need any help you can click on the green reference button top right. To save your CAM click on the disk symbol.
Please zoom out so that you can see the entire drawing window of the CAM. You can now close this dialogue window and start drawing. Have fun.
